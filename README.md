# Habit Tracker

A Clean-Architecture-based Swift Habit Tracker demonstrating domain logic separation, async/await flows, and unit testing.

## Architecture
- Domain
- Data
- Presentation
- Dependency Injection

## Features
- Add/Edit/Delete habits
- Daily Completion
- Streak tracking
- Offline persistence
- Unit tests

## Tech
Swift, SwiftUI, Core Data, XCTest

## How to run
1. Clone the repo
2. Open in Xcode
3. Run on Simulator
